オプションで渡した正規表現を基に値を検証します。

## オプション項目

| オプション | 型     | 初期値 | 説明 |
|------------|--------|--------|------|
| pattern    | regexp |        | 値を検証する正規表現です。必須です |

## 利用できるタグ

| タグ種別               | 補足 |
|------------------------|------|
| input[type="text"]     |      |
| input[type="password"] |      |
| input[type="search"]   |      |
| input[type="tel"]      |      |
| input[type="number"]   |      |
| textarea               |      |
